Отправка данных о портфелях в MQTT:

Брокерский счёт:

- `tinkoff/brk/buy`
- `tinkoff/brk/total`
- `tinkoff/brk/profit`
- `tinkoff/brk/percent`

ИИС:

- `tinkoff/iis/buy`
- `tinkoff/iis/total`
- `tinkoff/iis/profit`
- `tinkoff/iis/percent`

Курсы акций общие, падают в топики типа:

- `tinkoff/stocks/YNDX/buy`
- `tinkoff/stocks/YNDX/total`
- `tinkoff/stocks/YNDX/profit`
- `tinkoff/stocks/YNDX/percent`
